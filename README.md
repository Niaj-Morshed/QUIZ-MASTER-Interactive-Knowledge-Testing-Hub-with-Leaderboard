🧠 QUIZ MASTER – Interactive Knowledge Testing Hub with Leaderboard
--------------------------------------------------------------------
Live Demo: https://niaj-morshed.github.io/QUIZ-MASTER-Interactive-Knowledge-Testing-Hub-with-Leaderboard/


🚀 Overview:
-------------
Quiz Master is a high-end, dynamic quiz application designed to challenge users across various categories
like Web Development, Science,and General Knowledge. With features like real-time scoring, a global 
leaderboard, and data visualization, it offers a gamified learning experience with a sleek, modern UI.


✨ Key Features:
-----------------
1. 📂 Category-Based Quizzes
     * Diverse Subjects: Choose from multiple categories including HTML, CSS, JavaScript, and more.
     * Dynamic Loading: Questions are loaded dynamically based on the selected category to keep the content fresh.

2. 🏆 Global Leaderboard System
     * Top Performers: A dedicated leaderboard displaying the top 5 scores with Gold, Silver, and Bronze badges.
     * Real-time Ranking: Ranks are updated instantly after completing a quiz using LocalStorage.

3. 📊 Visual Score Analytics
     * Interactive Charts: Integrated with a custom Score Chart to visualize your performance history.
     * Performance Tracking: Helps users see their progress over time through graphical representation.

4. ⏱️ Real-time Timer & Progress
     * Countdown System: Each question comes with a time limit to test speed and accuracy.
     * Progress Bar: A visual bar indicating how many questions are remaining in the session.

5. 🌙 Dual Theme System (Dark/Light Mode)
     * Custom Aesthetics: Seamlessly toggle between a vibrant Light mode and a professional Dark mode.
     * Theme Persistence: The app remembers your preferred theme using browser storage.

6. 🎨 Premium UI/UX Design
     * Gradient Aesthetics: Beautiful linear-gradient buttons and card layouts for a tech-savvy look.
     * Smooth Animations: Includes fade-in effects and smooth transitions between quiz states.
     * Confetti Celebration: Animated feedback for high scores to enhance user engagement.

7. 💾 Local Data Persistence
     * Never Lose Progress: All quiz results, user names, and settings are saved locally, ensuring data stays intact after a refresh.



🛠️ Technologies Used:
----------------------
1. Core Stack
     * HTML5: Semantic structure for a clean and accessible user interface.
     * CSS3: Advanced styling with Flexbox, CSS Grid, and custom Keyframe animations.
     * JavaScript (ES6+): Powering the quiz logic, timer, scoring system, and DOM updates.

2. Libraries & APIs
     * LocalStorage API: For managing leaderboard data and user preferences without a backend.
     * Custom Canvas API: Used for drawing interactive score charts and performance graphs.
     * Google Fonts: Utilizing 'Arial' and professional sans-serif fonts for high readability.

📲 Fully Responsive Experience:
     * Cross-Device Compatibility: Perfectly optimized for Desktops, Tablets, and Mobile phones.
     * Touch Optimization: Large, easy-to-click buttons designed for a smooth mobile quiz experience.
